CHANGE NOTES

현 디자인 → Phase02 변경 5개

  • 01상단 안내 문구
    현 카트 상단의 핑크 안내 박스 → 문구만 교체
    AS-IS"配送タイプごとにお届け時期が異なるため、別々にご注文ください"
    TO-BE"배송유형별로 따로 배송됩니다. 모든 상품을 한 번에 주문할 수 있습니다"
  • 02그룹별 주문 버튼 제거
    현 디자인의 각 그룹 끝 "그룹별 주문 버튼" 3개 모두 제거
    AS-IS「即日発送(国内)の商品を注文する」 + 「即日発送(韓国)の~」 + 「通常配送の~」 (각 그룹별 1개씩)
    TO-BE제거 — 합결제는 통합 주문 버튼 1개만
  • 03그룹 소계·송료 표시
    정책서 정합 — 그룹 단위 소계는 유지 (배송유형별로 보여주는 게 옳음)
    유지「選択商品の金額 ¥X + 送料 ¥X = ¥X(目安)」 + 「あと6,000円で送料無料」
  • 04하단 안내 문구 제거
    注文する 버튼 위의 빨간 경고 안내문 제거
    AS-IS"※異なる配送タイプの商品は同時に注文できません"
    TO-BE제거 — Phase02는 다른 배송유형 동시 주문 가능
  • 05통합 주문 버튼 활성화
    하단 注文する 버튼 회색 비활성 → 검정 활성. 모든 그룹 합산 주문
    AS-IS회색·비활성 (한 그룹만 선택해야 그룹별 버튼이 활성)
    TO-BE검정·활성 → 선택 상품 전체 합산해서 주문서로 이동

유지되는 영역 (변경 없음)

  • 헤더 (← / カート / 🏠 ⌕)
  • 탭 (すべて / 国内 / 韓国 / 通常) — 필터 역할로 유지
  • 카트 액션 (カートを空にする / クーポン適用後の価格を表示 토글)
  • 배송유형별 그룹화 (그룹 헤더·아이콘·배송 시기 안내)
  • 상품 카드 구조 (체크박스·이미지·이름·태그·옵션·수량·금액)
  • セール終了まで 카운트다운
  • 그룹 단위 「あと6,000円で送料無料」 안내
  • 최종 합계 영역 + 핑크 무료배송 배너 + 주석 (※)
  • あなたへのおすすめアイテム 캐러셀

POLICY REFERENCE

정책서 "장바구니 정책 — 배송유형 그룹 유지" + "주문 방식 변경 — 통합 주문 버튼 1개 제공" + 본 문서 D02·D17·D18·D23·§3 영역 정합. 명칭 ShippingGroup (v0.34) 적용. 그룹 소계의 송료는 안내 목적이며 최종 결제 금액은 주문서에서 D23 (쿠폰 적용 후 결제 금액 기준) 무료배송 판정 결과로 확정.
